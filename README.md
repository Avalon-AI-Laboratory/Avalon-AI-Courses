# Avalon-AI-Courses

This repository collects various courses related to Artificial Intelligence, Machine Learning and Software Engineering. Below are some of the key topics covered:
- Math foundations for Machine Learning (calculus, linear algebra, probability, and statistics)
- Deep Neural Networks (concepts and hands-on labs)
- Natural Language Processing (fundamentals and language understanding)
- Computer Vision (CNN, Auto Encoder, GAN, and Diffusion)
- Docker (Basic and Study Cases)
